一、基础设定
全称：Bosch SBTI 博世职场人格测试
定位：仿 QBTI 沙雕人格测试、博世人专属玩梗版
题量：21 题（四选一）
维度：4 字母四维 → 组合出16 种博世人格
风格：贴合博世工厂 / 研发 / 质量 / 采购 / 行政 / 销售真实职场梗

二、四维定义（对标 MBTI/QBTI，博世定制）
第一维：处事模式「严谨 R / 随性 S」
R 严谨德系：按流程、留痕迹、版本齐全、数据说话
S 随性摆烂：差不多就行、能过就过、不爱走繁文缛节

第二维：工作态度「内卷 J / 摸鱼 M」
J 内卷战神：主动加活、自学提升、项目追到底
M 佛系摸鱼：到点下班、多一事不如少一事、不主动卷

第三维：人际风格「直给 D / 圆滑 Y」
D 直给硬刚：有问题当场说、讲事实不绕弯、不怕得罪人
Y 圆滑客套：会做人、会打圆场、不得罪人、懂得迂回

第四维：价值取向「质量 Q / 将就 O」
Q 质量执念：零缺陷、抠细节、返工也要做到标准
O 将就凑活：小瑕疵无视、能用就行、不愿反复打磨

人格格式：第一位 R/S + 第二位 J/M + 第三位 D/Y + 第四位 Q/O
例：RJDQ、RMYQ、SJDO…… 共 16 种

四、计分规则（四选一均衡版 严格1/2概率）
选项对应维度，全局统一21题通用，4选项2:2对半分配，各维度倾向概率均等1/2

第一维 处事模式 严谨R / 随性S
选 A=R  选 B=R  选 C=S  选 D=S

第二维 工作态度 内卷J / 摸鱼M
选 A=J  选 D=J  选 B=M  选 C=M

第三维 人际风格 直给D / 圆滑Y
选 A=D  选 C=D  选 B=Y  选 D=Y

第四维 价值取向 质量Q / 将就O
选 A=Q  选 D=Q  选 B=O  选 C=O

统计规则：
每题选定A/B/C/D后，按上述四维映射各得出一个字母，
按顺序拼接：第一位R/S + 第二位J/M + 第三位D/Y + 第四位Q/O，
拼成4字母人格码即可对照解读。

概率均衡说明：
每个维度均为2个选项对应正向、2个选项对应反向，
R:S、J:M、D:Y、Q:O 天然1:1分布，
每个字母出现概率严格等于1/2，无偏向无倾斜。
