# Bosch SBTI 博世职场人格测试

纯前端趣味版博世职场人格测试。通过 21 道四选一题目，在浏览器中即时计算并展示 16 种博世职场人格类型。

所有计分逻辑和题库数据均在客户端（`quiz.js`）中完成，**不依赖后端服务**，**不留存任何用户数据**。

## 功能特性

- **21 道情境题**（四选一）：贴近博世工厂 / 研发 / 质量 / 采购 / 行政 / 销售真实职场梗
- **16 种人格类型**：四维组合（严谨/随性 × 内卷/摸鱼 × 直给/圆滑 × 质量/将就）
- **纯客户端计分**：答题后即时显示人格类型、描述、四维分布条形图，无需服务器端处理
- **响应式设计**：适配桌面和移动端
- **无数据留存**：所有数据在浏览器中处理，不写数据库、不记录 IP、不生成任何日志
- **页面访问统计**：基于 `localStorage` 的页面刷新计数，每次刷新页面自动累计

## 快速开始

### 本地开发

直接用浏览器打开 `index.html` 即可，无需启动任何后端服务。

或使用任意静态文件服务器：

```bash
# Python 内置 HTTP 服务器
python -m http.server 8000

# Node.js（需安装 serve）
npx serve .
```

### GitHub Pages 部署

项目已配置 GitHub Actions 自动部署。推送到 `main` 分支后会自动部署到 GitHub Pages。

**手动配置：**

1. 在仓库 Settings → Pages 中，Source 选择 **gh-pages** 分支，根目录
2. 推送代码到 `main` 分支，Actions 会自动构建部署
3. 访问 `https://<用户名>.github.io/<仓库名>/` 即可

## 计分规则

### 四维定义（21 题通用，A/B/C/D 四选一）

| 维度 | 选项 A | 选项 B | 选项 C | 选项 D |
|------|--------|--------|--------|--------|
| 第一维：处事模式 | R 严谨德系 | R 严谨德系 | S 随性摆烂 | S 随性摆烂 |
| 第二维：工作态度 | J 内卷战神 | M 佛系摸鱼 | M 佛系摸鱼 | J 内卷战神 |
| 第三维：人际风格 | D 直给硬刚 | Y 圆滑客套 | Y 圆滑客套 | Y 圆滑客套 |
| 第四维：价值取向 | Q 质量执念 | O 将就凑活 | O 将就凑活 | Q 质量执念 |

### 人格码生成

统计四个维度各自得票，每维取票数高的字母，拼接成 4 字母人格码。

示例：`R`（严谨）+ `J`（内卷）+ `D`（直给）+ `Q`（质量）= `RJDQ`

概率均衡：每个维度均为 2 个选项对应正向、2 个选项对应反向，各字母出现概率严格为 1/2。
